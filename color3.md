# color3

color3 module

- [new](#new)

## new

```lua
color3.new(r, g, b)
```

Used to create a new color3.

| Input | Type | Description |
| --- | --- | --- |
| `r` | `number` | r element in range of 0 to 1 |
| `g` | `number` | g element in range of 0 to 1 |
| `b` | `number` | b element in range of 0 to 1 |

| Output type | Description |
| --- | --- |
| `color3` | color |
